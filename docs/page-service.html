Мультимедиа (AV)

Весь комплекс решения задач под «ключ» объектов любой сложности с циклом Генпроектировщик-Генподрядчик: Комплекс мультимедийных систем (КМС); Комплекс информационных систем (КИС); Система аудио-видео сопровождения (САВС); Мультимедийные системы (ММС). Интеграция аппаратно-программных комплексов с управлением инженерными системами объектов, системами управления производств на предприятиях.
